# MatrixCalculator
A matrix calculator made in Python! You can add, subtract, multiply and get identity matrices easily.
Made with study purposes.

Options: 
| 1 - [+] Add matrices |
| 2 - [-] Subtract matrices |
| 3 - [x] Multiply matrices |
| 4 - [I] Identity matrix |
| 5 - Exit program |

Thanks in advance for reading.
